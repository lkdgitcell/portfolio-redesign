Projet Bibliothèque

SLAM 2023-2024

Projet BTS SIO SLAM

Description

Projet de développement web réalisé dans le cadre de la matière " slam" (Solutions Logicielles et Applications Métier) du BTS SIO. L'application permet de gérer une bibliothèque : prêts, retours, consultation du catalogue et gestion des membres.

C'était mon premier projet majeur en développement web, l'occasion d'apprendre les bases du PHP et de MySQL, ainsi que la structure d'une application web complète.

Technologies utilisées

PHP MySQL HTML5 CSS3 JavaScript Bootstrap

Fonctionnalités

  • Interface d'authentification pour librarians et membres
  • CRUD complet pour le catalogue de livres
  • Gestion des prêts et retours avec dates d'échéance
  • Système de recherche avancée par titre, auteur,isbn
  • Gestion des membres (inscription, modification, désactivation)
  • Dashboard estadísticas : prêts en cours, retards, popular books
  • Notifications pour les retours快要到期
  • Export catalog en CSV

Base de données

Base MySQL avec tables : livres, membres, prêts, auteurs, catégories, exemplaires. Relations normalisées avec contraintes d'intégrité référentielle.
