Pare-feu OPNsense

SISR 2024-2025

Projet BTS SIO SISR

Description

Projet de mise en place d'un pare-feu OPNsense dans le cadre de la matière "Cybersécurité" du BTS SIO SISR. L'objectif était de créer une infrastructure réseau sécurisée avec un firewall performant, un système de détection/prévention d'intrusion, un VPN pour l'accès à distance et une segmentation réseau via VLANs.

OPNsense est une distribution BSD open-source basée sur HardenedBSD, offrant une alternative robuste aux solutions commerciales comme pfSense. Le projet a été réalisé sur mon Home Lab avec du matériel dédié.

Technologies utilisées

OPNsense Suricata WireGuard VLAN HAProxy Unbound DNS

Configuration réalisée

  • Installation et configuration initiale d'OPNsense sur serveur dédié
  • Configuration des règles de pare-feu (WAN, LAN, DMZ)
  • Mise en place de Suricata en mode IDS/IPS avec règles ET Open
  • Configuration VPN WireGuard pour accès distant sécurisé
  • Segmentation réseau VLAN (IoT,침, guests, servers)
  • Configuration HAProxy pour le reverse proxy interne
  • Mise en place DNS filtrant avec Unbound
  • Test et optimisation des performances réseau
