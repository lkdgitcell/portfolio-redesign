Proxmox Backup Server

SISR 2024-2025

Projet BTS SIO SISR

Description

Projet de mise en place d'un serveur de sauvegarde pour mon environnement Proxmox. L'objectif était de disposer d'une solution de sauvegarde fiable et sécurisée pour mes machines virtuelles, avec chiffrement des données et automatisation complète.

Proxmox Backup Server (PBS) est une solution open-source de sauvegarde adaptée aux environnements virtualisés Proxmox. Elle offre des fonctionnalités avancées comme la déduplication, le chiffrement et la réplication distante.

Technologies utilisées

Proxmox Backup Server Proxmox VE ZFS Chiffrement AES-256 PBS Client

Configuration réalisée

  • Installation Proxmox Backup Server sur serveur dédié
  • Configuration stockage ZFS avec compression et déduplication
  • Création datastore pour sauvegarde machines virtuelles Proxmox
  • Configuration du client PBS sur les nœuds Proxmox
  • Mise en place politique de sauvegarde (quotidienne, hebdo, mensuelle)
  • Configuration chiffrement AES-256 pour données sensibles
  • Automatisation avec PBS sync et tâches planifiées
  • Test de restauration complète et vérification intégrité
  • Configuration alertes et monitoring espace disque
