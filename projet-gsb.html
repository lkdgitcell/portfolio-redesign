Projet GSB – Cisco

SISR 2024

Projet BTS SIO SISR

Description

Projet de simulation réseau avec Cisco Packet Tracer dans le cadre de la matière "Communication et réseaux" du BTS SIO SISR. L'entreprise fictive GSB (Laboratoire Pharmaceutique Galaxy) necesitaba une infraestructura de red sécurisée con múltiples servicios et une segmentation adecuada.

Packet Tracer permet de simuler des équipements réseau Cisco dans un environnement virtuel. C'est un outil pédagogique essentiel pour apprendre les bases du networking.

Technologies utilisées

Cisco Packet Tracer Switch Cisco Router Cisco VLAN Routage inter-VLAN ACL

Architecture réseau

  • Configuration de 3 bâtiments avec switches gérés Cisco 2960
  • Segmentation VLAN par service : Direction, Recherche, Production, Administratif
  • Routage inter-VLAN avec router-on-a-stick
  • Mise en place DHCP relay pour attribution IPs dynamique
  • Configuration ACL pour restriction accès entre VLANs
  • Implémentation NAT/PAT pour accès Internet
  • Configuration routeur border pour connexion FAI
  • Test连通性 et dépannage réseau
