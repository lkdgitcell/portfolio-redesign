Alternance APYKA Informatique

En cours 2025-2026

Technicien IT · Alternance

Description

Dans le cadre de ma formation BTS SIO SISR, je effectue mon alternance chez APYKA Informatique en tant que technicien informatique. Cette alternance me permet de mettre en pratique mes connaissances en infrastructure tout en acquérir de nouvelles compétences professionnelles.

APYKA Informatique est une entreprise de services spécialisée dans le support informatique pour les professionnels. Mon rôle principal consiste à assurer le support technique de niveau 1 et 2 auprès des clients, ainsi que l'administration des infrastructures Windows Server.

Technologies utilisées

Windows Server 2019/2022 Active Directory Exchange Server SharePoint M365 VPN Ticketing

Missions principales

  • Support IT helpdesk : diagnostic et résolution de pannes matérielles et logicielles
  • Administration Windows Server : gestion des utilisateurs, permissions et stratégies de groupe
  • Gestion Active Directory : création de comptes, attribution de droits d'accès
  • Déploiement postes de travail : installation et configuration de Windows 10/11
  • Maintenance matérielle : intervención sur parc informatique clients
  • Documentation technique : rédaction de procédures et guides utilisateurs
